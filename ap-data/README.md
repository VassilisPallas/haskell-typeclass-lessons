# ap-data

A collection of data types implemented as part of lessons to the audio-platform
team.

### Dependencies
- [Haskell Stack](https://docs.haskellstack.org/en/stable/README/)

From this directory run
| Command | Script      |
| ------- | ----------- |
| Build   | stack build |
| Test    | stack test  |
| Clean   | stack clean |
| Purge   | stack purge |
